## 🛠️ Tech Stack
- React + Vite
- Tailwind CSS
- Capacitor (Android)
- Netlify (Hosting)

## 🚀 Getting Started

### Install dependencies
```bash
npm install
```

### Run in browser
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Run on Android
```bash
npm run build
npx cap sync android
npx cap open android
```

## 📁 Project Structure
- `src/`        → React source code
- `android/`    → Android native project
- `dist/`       → Production build
